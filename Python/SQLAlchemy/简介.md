# 什么是SQLAlchemy
灵活性的Python库SQLAlchemy，可以跨越关系型数据库与传统编程直接的鸿沟。SQLAlchemy允  
许你使用原始的SQL语言查询，更鼓励使用更高级的Pythonic展示方式和更友好的方法实现数据的查询和更新。它提供工具让你一次性把应用的类和对象  
映射成数据库的表，然后就可以不再理会表结构了，还可以让你不断优化模型的性能。

 优势
 - 迁移库, 例如Oracle库迁移到其它库
 - 防止SQL注入
 - 抽象数据模型
