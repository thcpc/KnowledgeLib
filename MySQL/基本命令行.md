# 连接数据库
```shell
mysql -uroot -p123456
```

# 修改数据库密码
```sql
update mysql.user set authentication_string=password('123456') WHERE user = 'root' AND host = 'localhost'
```

# 刷新
```sql
flush privileges;
```

# 显示所有数据库
```sql
show databases;
```
# 使用某个库
```sql
use 库名;
```

# 显示所有表
```sql
show tables;
```

# 显示表的描述信息
```sql
describe 表名;
```

# 查看表的DDL
```sql
SHOW CREATE TABLE 表名;
```
